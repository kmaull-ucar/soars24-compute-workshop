Tools, Guides and Inspiration#

Productivity#

  • [browser extension] uBlock origin (uBO): ads are productivity busters whether we agree or not – save the bandwidth with this excellent free and open source tool and consider the hit ads make to your online productivity and privacy “… uBO’s primary goal is to help users neutralize [these] privacy-invading methods in a way that welcomes those users who do not wish to use more technical means.”

  • [text editor] vscode (Microsoft download, non-free license) or vscodium (telemetry free, open source license): everyone needs a good text editor

Python and Jupyter#

Jupyter Notebooks and Jupyter Books#

  • Project Jupyter

    • Binder: launch reproducible notebooks interactively online

    • Jupyter nbviewer: make your notebooks render better on the web with nbviewer

    • Gallery of Interesting Jupyter Notebooks: a large compendium of notebooks of all kinds (many on Github)

  • Jupyter Books: take your notebooks to the next level and set your project up for publication-grade online and offline presentation

    • Gallery of Jupyter Books

Core Geoscience Python Libraries#

  • MetPy: a collection of tools in Python for reading, visualizing and performing calculations with weather data

  • EarthPy: makes it easier to plot and manipulate spatial data in Python

  • GeoPandas: makes working with geospatial data in python easier

  • Rasterio: reads and writes GeoTIFF and other gridded/raster data and provides a Python API based on N-D arrays

  • ArcPy: provides support for ArcGIS manipulation

Research Discovery#

There are a number of research discover tools that you may have access to on your campus, but here are some that do not require subscriptions (to search):

  • SemanticScholar: exceptional tool for modern discovery tactics

  • Dimensions.ai: excellent tool for additional discovery; many additional features for discovering domain research

  • GoogleScholar: classic tool, but has poor integration with DOI importation and can provide spurious results; often useful for finding author PDFs when publisher version is paywalled

  • Awesome Open Science (Github): a massively list of open source, open science tools, links and stuff

  • Awesome Open Geoscience: an excellent geo-specific resource that you cannot miss out on

Scientific Workflows#

  • Open Science Framework: designed to promote computational workflows, transparency and collaboration across many disciplines

  • Deepnote: share notebooks more effectively (commercial tool)

AI Tools#

There are a number of emerging AI tools, not the least of which is ChatGPT. Here are others you may also want to know about:

  • phind: an AI search engine for developers and great resource to help with your coding experiences

  • autogpt: AI automation using GPT technology stack

  • privateGPT: feeling like running your own LLM? Here’s your chance to have a conversation with your own data DIY-style

  • open-webui: user-friendly open source web-based UI you can run locally for your local LLMs; it’s magic, but not really

Citation and Bibliographic Management#

  • Zotero: free, very useful and low barrier bibliographic management

    • use Zotero browser plugin: essential, must-have tool!

  • Mendeley: excellent bibliographic and research management platform

Code Management#

  • git: the open-source distributed revision control system for everything technical and upon which many front-ends are built; learning it will change your life

  • Github: enterprise-grade Git hosting; the “OG” of open source Git hosting small and large

  • Gitlab: enterprise-grade Git hosting; can be Linux self-hosted, but without some effort

  • Bitbucket: one of the big three and still an excellent alternative

  • Codeberg: privacy-respecting open source code hosting, all repos are free and non-private

  • Gitea: feeling like self-hosting? – this open source project is an awesome lightweight alternative for DIY solo and small teams and gives you full control of your code hosting (not for the faint at heart)

Writing#

  • Distraction-free writing: theory, practice and some fun

  • Markdown: easy, distration-free markup writing for durability and portability; takes minutes to learn

  • Pandoc: free, open source conversion tool “swiss army knife” of document formats

  • Scientific writing in Markdown

    • Sustainable Authorship in Plain Text using Pandoc and Markdown by Dennis Tennen and Grant Wythoff

    • RMarkdown for writing reproducible scientific papers by Mike Frank & Chris Hartgerink

    • Awesome Scientific Writing

  • \(\LaTeX\): a rabbit hole perhaps not worth going down … but with a cheat sheet you can be used to seemlessly integrate mathematics into your Markdown writing

    • LaTeX Math cheat sheet for undergraduates

    • OverLeaf : helps overcome some of the LaTeX barriers and encourage collaboration

Data Science and Data Analytics#

Training#

Python and Geoscience

  • Project Pythia: education and training hub for the Geoscience Python community – a truly wonderful resource that you must know about

  • Software Carpentry: begin the journey into a more experienced and confident software developer

    • Version Control with git

    • Programming with Python

  • Data Carpentry: level up your data science and analytics skill with these free courses and resources

    • Introduction to Geospatial Concepts:

  • EarthLab: free, online courses to learn how to use earth science data in Python and R

    • Earth Data Analytics Online Certificate: an excellent resource to be aware of even if you don’t want to get the certificate!

  • PyEarth: A Python Introduction to Earth Science (Course)

Machine Learning and AI

  • Andrew Ng’s Machine Learning Course Notebooks: collection of excellent notebooks on introductory machine learning

  • Earth Artificial Intelligence: great book on AI in Geosciences

Tools#

  • OpenRefine: free and open source data cleaning and transformation
